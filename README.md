# Daikin IoT Project (ESP8266 + Blynk + IR Control)

Sistem kontrol AC Daikin berbasis IoT menggunakan ESP8266 dengan integrasi Blynk, sensor suhu & kelembapan, serta kontrol IR.

## 🔥 Fitur Utama
- Kontrol AC Daikin via IR transmitter
- Monitoring suhu & kelembapan (DHT22)
- Kontrol jarak jauh via Blynk IoT
- Kontrol fan & aktuator (L298N)
- Indikator buzzer
- Logika otomatis (threshold / fuzzy-ready)

## 🧠 Komponen Hardware
- ESP8266 (NodeMCU / Wemos D1 Mini)
- DHT22
- IR LED transmitter
- IR receiver (learning remote)
- L298N motor driver
- Fan + humidifier
